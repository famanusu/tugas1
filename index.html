Perhitungan Koefisien Regresi dan Evaluasi Model

1. Persamaan Regresi Linear

Model regresi linear dinyatakan sebagai:

y = β₀ + β₁x

ΣX = 30, ΣY = 375, ΣXY = 2450, ΣX² = 220, n = 5

Hitung β₁:

β₁ = (nΣXY - ΣXΣY) / (nΣX² - (ΣX)²)

β₁ = (5×2450 - 30×375) / (5×220 - 30²)

β₁ = (12250 - 11250) / (1100 - 900) = 1000 / 200 = 5

Hitung β₀:

β₀ = (ΣY - β₁ΣX) / n

β₀ = (375 - (5 × 30)) / 5

β₀ = (375 - 150) / 5 = 225 / 5 = 45

Persamaan Regresi:

Y = 45 + 5X

3. Evaluasi Model (R²)

Hitung R²:

R² = 1 - (Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²)

R² = 1 - (0 / 1000) = 1

4. Kesimpulan

  • Model regresi: Y = 45 + 5X
  • Koefisien β₁ = 5 → Setiap tambahan 1 jam belajar meningkatkan nilai ujian sebesar 5 poin.
  • R² = 1 → Model ini sangat akurat dalam menjelaskan hubungan antara jam belajar dan nilai ujian dalam dataset ini.
